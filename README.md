> **[annexa](https://irfanm46.github.io/annexa)** — The ISO 27001:2022 toolkit that turns 93 Annex A controls into actionable tools. Built for engineers and GRC teams who need to implement compliance, not just document it.

<div align="center">

| Feature | Purpose |
|---------|---------|
| Compliance Heatmap | Track all 93 controls at a glance |
| Risk Register | Score risks, sort by severity, export for auditors |
| SoA Generator | Statement of Applicability — audit-ready PDF |
| Threat Mapper | Real attack scenarios mapped to control gaps |
| Posture Score | ISMS maturity radar across 4 Annex A domains |
| Evidence Wizard | Exact checklist of what auditors will ask for |
| Code Snippets | Implementation code per control for dev teams |
| 2013 → 2022 Diff | Full migration map for transitioning orgs |

**Free · Offline · No signup · Open Source**
