## Featured Project — annexa

<div align="center">

[![annexa](https://github-readme-stats.vercel.app/api/pin/?username=irfanm46&repo=annexa&theme=tokyonight&hide_border=true&bg_color=0a0a0f&title_color=00e5ff&text_color=cbd5e1&icon_color=00e5ff)](https://github.com/irfanm46/annexa)

</div>

**[annexa](https://irfanm46.github.io/annexa)** maps every ISO 27001 Annex A control to real deployable code.

| Tool | What it does |
|------|-------------|
| Compliance Heatmap | Visual grid of all 93 controls — click to mark status |
| Code Snippets | Terraform · GitHub Actions · Bash · Python per control |
| Risk Register | Auto-scored, severity-sorted, CSV + PDF export |
| Threat Mapper | Pick an attack — see exactly which controls you're missing |
| SoA Generator | Audit-ready Statement of Applicability in 5 minutes |
| Posture Score | Radar chart of ISMS maturity across 4 domains |
| Evidence Wizard | Per-control checklist of what auditors will ask for |
| 2013 → 2022 Diff | Full migration map between ISO versions |

**Zero dependencies · Works offline · No signup · 144KB single file**
